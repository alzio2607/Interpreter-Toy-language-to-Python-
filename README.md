# Interpreter-Toy-language-to-Python-
A course based project on Python language. Given a toy language code , the interpreter converts and runs this code using Python

Example Toy Code :

n=(1+1)+ (1*2);
fact=1;
count=n;
while count!=0 do:
fact =fact*count;
count =count-1;
end;
print "Factorial is";
print \n;
print fact;
print fact*4;
